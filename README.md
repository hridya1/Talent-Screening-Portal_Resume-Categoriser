# Talent Screening Portal - Resume Categoriser
AI powered resume Talent screening Portal and JD matching project using NLP and Machine Learning, Streamlit.


## Features
- Resume Upload (PDF/DOCX)
- Candidate Classification
- AI-Based Resume Screening
- Job Description Matching
- Candidate Ranking Dashboard

## Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn
- NLP
- GitHub
